# Validator of Ethereum-speicific Ledger Plugin 

## Assumption
- A ledger (Ethereum node) to connect is already running
- Specify URL of the ledger node to connect to in "provider" of "PluginConfig.js"

## Execution
```
./setup.sh
cd core
node ./bin/www.js 
```
